Deployment notes for dogs.paksarzameenwfo.com

- Set `NEXT_PUBLIC_API_BASE` to the parent API base, e.g. https://paksarzameenwfo.com
- Configure `DATABASE_URL` only if you plan to run server-side DB routes from this project.
- On Vercel, set the project domain to `dogs.paksarzameenwfo.com` and add `robots.txt` + `sitemap.xml` in `public/`.
