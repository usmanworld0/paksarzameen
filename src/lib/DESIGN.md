# DESIGN SYSTEM — Premium Minimal UI

## Design Goal

Create a clean, modern, premium interface inspired by Apple and luxury brands.
Focus on simplicity, whitespace, typography, and visual hierarchy.
DO NOT change functionality — only improve visual design.

---

## Core Principles

* Design in grayscale first, add color last
* Use fewer elements, not more
* Prioritize whitespace over density
* One primary action per section
* Avoid visual clutter at all costs

---

## Layout & Spacing

Use a consistent spacing scale:

4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px, 96px

Rules:

* Sections: 80px–120px vertical spacing
* Cards: 24px–32px padding
* Elements: maintain consistent gaps
* Related items are closer together
* Unrelated sections have large separation

---

## Typography

Use a clean sans-serif font:

* Inter / SF Pro / Helvetica Neue

Hierarchy:

* H1: 48–64px (bold)
* H2: 32–40px (semibold)
* H3: 20–24px (medium)
* Body: 16–18px (regular)

Rules:

* Avoid too many font weights
* Do NOT use light gray text on colored backgrounds
* Use size and weight for hierarchy, not color

---

## Colors

Minimal palette:

* Background: #FFFFFF or off-white
* Primary text: #111111
* Secondary text: #666666
* Accent: ONE color only (e.g. #000000 or subtle gold)

Rules:

* Avoid multiple bright colors
* Avoid gradients unless extremely subtle
* Use color only for emphasis

---

## Buttons

Style:

* Large padding (12px–16px vertical, 20px–28px horizontal)
* Rounded corners (8px–12px)
* No borders or very subtle borders

Primary Button:

* Dark background
* White text

Secondary Button:

* Light background
* Dark text

Hover:

* Slight opacity or subtle shadow

---

## Cards & Containers

* Background: white

* Border: none or very subtle (#eee)

* Use soft shadows:
  box-shadow: 0 4px 20px rgba(0,0,0,0.05);

* Rounded corners: 12px–16px

* Generous padding

---

## Forms & Inputs

* Large input fields
* Minimal borders
* Use spacing instead of lines

Focus state:

* Subtle glow or darker border

---

## Navigation

* Keep minimal
* Reduce number of links
* Use spacing instead of separators
* Align everything perfectly

---

## Visual Hierarchy

Use ONLY these tools:

* Size
* Weight
* Spacing
* Contrast

DO NOT rely on:

* Bright colors
* Excessive icons
* Borders everywhere

---

## Images

* Use high-quality, large images
* Avoid cluttered visuals
* Maintain consistent aspect ratios

---

## Shadows & Depth

* Very subtle shadows only
* No heavy or dark shadows
* Create depth through spacing first

---

## What to Avoid

* Too many colors
* Small cramped layouts
* Inconsistent spacing
* Overuse of borders
* Tiny buttons
* Multiple competing elements

---

## Implementation Instructions for Codex

When modifying UI:

* Do NOT change logic or functionality
* Only adjust styles (CSS, layout, spacing)
* Keep components intact
* Improve readability and visual balance

---

## Expected Outcome

* Clean, calm interface
* Premium look and feel
* Apple-like simplicity
* Luxury-level spacing and typography
* No visual noise
